# Notes App/To-Do List Android App
1. This is a single-player game, a user gets 30 seconds to answer MCQs.
2. To start the game, the player has to tap on GO!.
3. Player has to calculate the correct answer as quickly as possible.
4. Once a player attempts a question, the next question comes.
5. Correct questions/ total attempted questions will be shown side by side.
6. Game stops after 30s,player can play again using play again button.
7. This project is implemented using the Java language.
